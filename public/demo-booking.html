üéØ Appointment Booking Demo - No Request Body

This demo shows how to book appointments using URL parameters instead of request body.

üìã Method 1: Query Parameters

Using GET /book?name=...&email=...&date=...&doctor=...

Example URLs:

Book for Alice Johnson Book for Bob Wilson

URL Structure:

/book?name=Patient Name&email=patient@email.com&date=YYYY-MM-DD&doctor=doctor-id

üîó Method 2: Path Parameters

Using GET /book/:name/:email/:date/:doctor

Example URLs:

Book for Maria Garcia Book for David Chen

URL Structure:

/book/Patient Name/patient@email.com/YYYY-MM-DD/doctor-id

‚úÖ Benefits of Removing Request Body

  • Simpler API: No need to parse JSON or form data
  • URL-based: Easy to test and bookmark
  • Cache-friendly: GET requests can be cached
  • Direct links: Can create direct booking links
  • No CORS issues: Simple GET requests work everywhere

üìù Frontend Code Examples

Query Parameters Approach:

// JavaScript example
const params = new URLSearchParams({
  name: 'John Doe',
  email: 'john@example.com', 
  date: '2025-08-25',
  doctor: 'dr-smith'
});

fetch(`/book?${params.toString()}`)
  .then(response => response.text())
  .then(result => console.log(result));

Path Parameters Approach:

// JavaScript example  
const name = encodeURIComponent('Jane Smith');
const email = encodeURIComponent('jane@example.com');
const date = '2025-09-01';
const doctor = 'dr-williams';

fetch(`/book/${name}/${email}/${date}/${doctor}`)
  .then(response => response.text())
  .then(result => console.log(result));
